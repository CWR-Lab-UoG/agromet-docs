    Automation and Task Scheduler

    Daily processing is orchestrated using Windows Task Scheduler. Each scheduled task launches a batch (.bat) file that in turn calls Python or MATLAB scripts.

    • Failures in one site do not block others
    • Logs are written for traceability and recovery
